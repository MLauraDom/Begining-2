What is the div tag and why is it important? What is the difference between div and span elements?

The div tag represents a container unit that can encapsulate other HTML elements and divide the document into sections or divisions.
This element is very important and is primarily used to group other HTML elements.
The div - element is an example of a block-level element and it starts on a new line and takes up the full width available.
Span is an inline element, it doesn’t start on a new line and only takes up as much width as necessary.

What is the purpose of semantic elements in HTML5?

Semantic element are HTML elements for text formatting with or without special meaning. They make the text Bold, Italic, subscript, superscript, strikethrough, etc.

Why are Meta Tags important and what can you do with Meta Tags?

Metadata is sometimes called ‘data about data’. The meta tag can also be used to imitate an HTTP response header (like redirection to a different page) and it can have important attributes like http-equiv and charset. Meta tags are the functionality that is hidden in your code.

How many list types do we have in HTML? Describe them.

In HTML we can distinguish three different list types:

  1. Ordered lists (numbered and represented with ol)
  2. Unordered lists (not numbered and presented with bullets for each list item by default and unordered lists are represented with ul )
  3. Definition lists (these lists are used when list items need a description and are represented with dl)

Could we make an entire page layout using tables? How?

Yes, it´s possible. As we know, tables are used to arrange data into rows, columns and cells. Tables were used to layout entire pages. Nowadays, that role has been taken over by CSS.

What is the purpose of forms in HTML?

The HTML form element defines a form that is used to collect user input. To accept input from a user, we must wrap all of your input fields inside a form tag. The form tag indicates where and how the user’s input should be sent.

What can you achieve using an iframe element?

An iframe enables us to embed a web page within another web page in the same way we add an image to the web page.

Why are comments important in well-written code?

As developers, we place comments into HTML pages to describe the page code, give information to our fellow developers or to our future selves. This part of code is ignored by the web browsers. Using HTML comments is excellent practice, especially in documents with a lot of code, because we can insert a description about the code, so our Partners can understand what we wanted to do, and also we can remember easily later what we wanted to do with this code.
